## Français


<a name="fr_pr"/>

### Présentation

Ce projet est un exercice de manipulation d'image en C.


<a name="fr_ut"/>

### Utilisation

Modification :

* Choisir l'image à importer dans main.c. L'image doit être au format ppm.
* Modifier l'image à l'aide des fonctions de image.c.

Compilation :

```
gcc src/main.c
a.exe
```
